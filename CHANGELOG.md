# Changelog

## [1.0.0] - 2026-06-26

### Added
- 自定义标题栏（无边框窗口），支持拖拽移动、最小化/最大化/关闭按钮
- 多标签页浏览：Ctrl+双击文件夹在新标签页打开，标签页关闭/切换/自动命名
- 标签页右键菜单：关闭其他、关闭右侧标签页
- ZIP/CBZ 文件作为文件夹浏览（`zip://` 路径方案）
- WebDAV 远程文件系统集成：添加/管理连接，远程文件浏览
- 文件操作：新建文件夹、新建文件、重命名、复制/剪切/粘贴、删除
- 上下文菜单（右键菜单）：文件/文件夹/驱动器专属操作项
- 列表视图增强：修改时间列、粘性表头、列宽拖拽调整、点击列排序
- 拖拽多选（蓝色选区框），Ctrl/Shift 多选
- 文件预览：图片编辑（旋转/裁剪）、Office 文档、PDF、代码高亮
- 构建自动版本号（`version.txt` + `build_number.txt`）

### Changed
- 侧边栏树状目录使用系统文件夹图标
- 面包屑分隔符改为 SVG 箭头图标，点击可编辑路径
- 工具栏精简：移除刷新和打开文件夹按钮
- 标题栏半透明毛玻璃效果（`backdrop-filter: blur`）
- 非最大化时标签页顶部留间隔

### Fixed
- 列表视图 `height: 100%` 不生效导致布局错乱
- ZIP 目录名解析为空的问题（`trim_end_matches('/')`）
- WebDAV 重连后会话 ID 不匹配
- `tabsStore` 初始化顺序导致的 `ReferenceError`
